ABOUT GAME DIFFICULTY

ADD
Level 1: [0 - 100] + [0 - 100]
Level 2: [10 - 1000] + [10 - 1000]
Level 3: [100 - 10000] + [100 - 10000]

SUBTRACT
Level 1: [0 - 100] - [0 - 100]
Level 2: [10 - 1000] - [10 - 1000]
Level 3: [100 - 10000] - [100 - 10000]

MULTIPLY
Level 1: [0 - 100] * [0 - 10]
Level 2: [10 - 1000] * [0 - 100]
Level 3: [100 - 10000] * [10 - 1000]

DIVIDE
Level 1: [0 - 100] / [1 - 10]
Level 2: [10 - 1000] / [1 - 100]
Level 3: [100 - 10000] / [10 - 100]
